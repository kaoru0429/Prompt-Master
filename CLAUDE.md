## 專案概述

Prompt Library Manager 是一個功能強大的桌面應用程式，提供：
- 訂閱 GitHub Prompt 集合（如 Awesome ChatGPT Prompts）
- 智慧變數引擎：將靜態 Prompt 轉為動態表單
- 多 AI 模型支援：OpenAI、Gemini、Claude、Ollama
- 本地優先：SQLite 儲存，完全掌控資料

## 技術棧

- **前端**: Electron + React + TypeScript + Zustand + shadcn/ui
- **後端**: Better-SQLite3 + Handlebars.js
- **AI 整合**: OpenAI、Gemini、Claude、Ollama
- **開發工具**: Vite + Jest + ESLint + Electron Builder
